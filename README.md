# DomestikAirLine
Airline management system for a fictional airline called Domestik, built with C# and database integration. Features include passenger, flight, and reservation management, secure authentication using hashed passwords, and a scalable architecture for future improvements and feature expansion.

# Domestik Airline Management System

## Features
- Passenger management
- Flight management
- Reservation system
- Secure password hashing
- Database integration
- Scalable architecture

## Technologies
- C#
- SQL Server / SQLite
- .NET
- Entity Framework (si usas)

## Future Improvements
- Role-based authentication
- Flight scheduling
- Payment integration
- Admin dashboard
- REST API support
